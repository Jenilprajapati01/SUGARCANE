# 🌾 Sugarcane EDA

## 📌 Overview  
This project performs **Exploratory Data Analysis (EDA)** on sugarcane data to extract insights and patterns that help understand key sugarcane characteristics.

## 🎯 Objectives  
- Explore sugarcane dataset features  
- Visualize distributions and relationships  
- Identify trends in important variables

## 📊 Dataset  
- Real sugarcane dataset (included in `Dataset/`)  
- Used for analysis and visualization

## 🛠 Tech Stack  
- 🐍 Python  
- 📓 Jupyter Notebook  
- 📦 Pandas, NumPy, Matplotlib, Seaborn

## 🔍 Methodology  
- Load and clean data  
- Summary statistics  
- Plot distributions and correlations  
- Draw insights from visual patterns
